# 企业官网静态页

这是一个可直接放到 GitHub Pages 的游戏与软件开发行业企业官网静态页。

## 文件

- `index.html`：官网首页
- `privacy.html`：隐私政策
- `terms.html`：服务条款
- `styles.css`：页面样式
- `favicon.svg`：浏览器标签页图标
- `assets/logo-mark.svg`：网站导航标识
- `assets/hero-studio.png`：首页视觉图

## 上线到 GitHub Pages

1. 新建 GitHub 仓库，把这些文件放到仓库根目录。
2. 进入仓库 `Settings` → `Pages`。
3. `Build and deployment` 选择 `Deploy from a branch`。
4. Branch 选择 `main`，目录选择 `/root`，保存。
5. 等待 GitHub Pages 部署完成。
